== Description ==

Suppa is Fast, Customizable, Powerfull and User-Friendly WordPress Mega Menu.
Suppa Work out of the box with almost any theme supporting Wordpress 3.0 Menus.
With suppa customization system no need to open css files , customize everything from the Plugin Options Page ( unlimited colors, background images , gradient … ) .
Suppa work with mobile devices ( iPhone , iPad , android … ) , responsive is “Suppa” priority .
All The Magic is Inside ( Appearence > Menus && Suppa Settings / Style Pages ).


**Features**

* Work With Wordpress 3.0 Menus
* Different Menu Types ( Dropdow, Logo )
* Link Settings ( Link Align , Upload Icon / Select Font Awesome Icon )
* Customize almost everything from plugin page
* User-Friendly
* Fully Mobile-Ready design adopts to any device !
* Retina Ready!
* Choose from click, hover or hover-intent trigger
* 3 Responsive Trigger Modes
* Scrolling Menu
* Layout ( Boxed / Wide )
* Ability to add background images to menu & submenus
* CSS3 ( border radius , box shadow , gradient )
* 100+ Google fonts!
* Ability to change the size and color of icons ( icons style system )
* 5 jQuery Animations ( Fade, Slide, Explode, Drop, Pulsate, None )
* Localisation Ready ( translate the plugin to your language )

== Installation ==

The easy way:

1. Go to the Plugins Menu in WordPress
2. Search for "Suppa Menu Free"
3. Click "Install"

The not so easy way:

1. Upload the `CTF_suppa_menu_free` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Manage your Menus using the 'Appearance > Menus' page
